# **📍 第一處：城市差異檢定（Kruskal-Wallis Test）**

## **１. 研究目的**

檢驗不同城市的學生在「Points、Badges、Accuracy」三個指標上是否存在顯著差異。
因為資料分佈明顯偏態、不同城市樣本量不同，因此不採用 ANOVA，而改用非參數的 Kruskal-Wallis H 檢定。

## **２. 方法**

將所有學生依據其所屬城市分組，分別比較下列變數：

* 總積分（Points）
* 獲得徽章數（Badges）
* 答題正確率（Accuracy）

Kruskal-Wallis 用來檢定多群體的中位數是否相同，p 值小於 .05 表示「至少有兩個城市之間存在差異」。

## **３. 結果與解釋**

| 指標         | H 值    | p 值                     | 解釋           |
| ---------- | ------ | ----------------------- | ------------ |
| points     | 745.31 | < 0.0000000000000000001 | 城市間積分分布有顯著差異 |
| badges_cnt | 657.12 | < 0.0000000000000000001 | 城市間徽章數有顯著差異  |
| accuracy   | 917.03 | < 0.0000000000000000001 | 城市間答對率有顯著差異  |

**結論：**
不同城市的學生在學習表現上確實存在差異，可能與地區資源、補教文化、學校師資或家庭背景等因素相關。

# **📍 第二處：行為模式差異檢定（Kruskal-Wallis / 二群比較）**

## **１. 研究目的**

比較四種行為模式（穩定型、爆發型、自學型、被動型）在學習成效上的差異。

## **２. 方法**

依照 Tableau 的邏輯，先用 Z score 建立三項標準化指標：

* 時數標準化（std_hours）
* 天數標準化（std_days）
* Points 標準化（std_points）

並用條件判斷分類為四種行為型態後：

再以「某群 vs 非某群」方式進行二群 Kruskal 檢定。

## **３. 結果與解釋**

以「Points、Badges、Accuracy」三項為比較變數，每一種行為類型的 p 值皆趨近於 0：

* **穩定型**學生的積分與徽章表現明顯高於非穩定型
* **爆發型**學生在短期內得到較高分，但天數較少
* **自學型**學生在無教師支援下仍達到高分，具有自主性
* **被動型**學生的總分、正確率明顯較低

**結論：**
四種類型之間的學習成效差異非常顯著，代表行為模式確實能有效刻畫學生的學習特徵。

# **📍 第三處：城市 × 行為模式（卡方檢定）**

## **１. 研究目的**

探討「城市」與「行為模式」之間是否存在關聯。
換句話說，不同城市的學生是否傾向出現不同的學習行為類型？

## **２. 方法**

建立交叉表：

```
城市 × （穩定型／爆發型／自學型／被動型）
```

使用卡方檢定比較各城市行為類型比例的差異。

## **３. 結果與解釋**

你得到：

* burst_type：p = 5.54e−35
* stable_type：p = 1.59e−227
* selflearn_type：p = 3.79e−15
* passive_type：p = 8.53e−145

所有 p 值皆遠小於 .05。

**結論：**
城市與行為模式之間具有顯著關聯。
表示：

> 不同城市的學生，在學習投入方式、操作習慣、回家練習量等行為上，存在明顯差異。

可能原因包含：

* 學校制度
* 地區教育資源
* 課後學習文化
* 家庭支援程度
* 補習班普及度
